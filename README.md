# Big-Salty-Keypad-5ROL
A 5 key Macropad with a rotary encoder, OLED display and 5 LEDs all using QMK firmware.

## Features
- Fully 3D Printable case
- 5 Programmable keys
- OLED display
- Rotary Encoder for volume control
- 5 LEDs for fun!

## CAD
This keypad comes together by sandwiching the PCB between the bottom and top parts of the case and fastening it together with 4 M3 screws. The 2 case peices are 3d printed as well as the rotary encoder's cap! I designed it in Fusion 360.

## Firmware
This keypad uses QMK firmware for everything!
- Encoder helps with volume control
- Programmable Keys using VIA
- OLED and LEDs!

## BOM
- 5x Cherry MX Switches
- 5x DSA Keycaps
- 4x M3x16mm SHCS Bolts
- 5x WS2812B LEDs
- 1x 0.91" 128x32 OLED Display
- 1x EC11 Rotary Encoder
- 1x XIAO RP2040
- 1x Case (2 printed parts)
